## MVP scope (summary)

- **Budget:** 220,000,000 VND · **Team:** 1 FE + 1 BE · **Duration:** 3 months
- **Screens:** 18 (10 Shop + 8 Admin)
- **Payment:** COD + bank transfer (no payment gateway fees in MVP)
- **Stack:** Quarkus modular monolith, Next.js 14, PostgreSQL, AWS (lean)
